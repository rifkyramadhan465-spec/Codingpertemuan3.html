# Codingpertemuan3.html
Formulir Pendaftaran

Formulir ini dibuat untuk mengumpulkan data diri dan informasi tambahan dari calon pendaftar. Pengguna diminta untuk mengisi seluruh bagian formulir dengan benar dan lengkap sebelum menekan tombol Kirim. Pada bagian Data Diri, peserta perlu menuliskan nama lengkap, alamat email aktif, serta membuat kata sandi yang aman. Kolom usia dan tanggal lahir dapat diisi sesuai data sebenarnya, sedangkan pilihan jenis kelamin dapat ditentukan dengan memilih antara laki-laki atau perempuan.

Selanjutnya, pada bagian Informasi Tambahan, pengguna dapat mencentang bidang minat yang sesuai seperti AI, Web, atau Game. Nomor telepon diisi dengan nomor yang masih aktif agar dapat dihubungi bila diperlukan. Apabila memiliki portofolio online, pengguna juga dapat menambahkan tautannya pada kolom yang telah disediakan. Di bagian akhir, terdapat kolom untuk mengunggah foto diri dalam format gambar (misalnya JPG atau PNG).

Setelah semua data terisi, pengguna dapat menekan tombol Kirim untuk mengirim formulir atau tombol Reset jika ingin menghapus seluruh isian dan memulai kembali. Pastikan seluruh kolom wajib sudah diisi agar proses pengiriman berhasil tanpa error.
